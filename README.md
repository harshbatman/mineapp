# Mine by MAHTO

Welcome to the **Mine** app repository. Mine by MAHTO is a premier full-stack construction, renovation, and service application built with React Native and Expo.

## Features

- **Construction**: Comprehensive services for residential and commercial projects.
- **Renovation**: Premium renovation solutions to transform spaces.
- **Service**: Reliable maintenance and repair services.

## Getting Started

This project is built with Expo. To run it locally:

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start the App**:
    ```bash
    npx expo start
    ```

3.  **Run on Device**:
    - Scan the QR code with the Expo Go app on Android or iOS (Camera app).
    - Or press `a` for Android emulator, `i` for iOS simulator.

## Tech Stack

- **React Native** (Expo SDK 52+)
- **Expo Router** (Future implementation planned)
- **Vector Icons** (@expo/vector-icons)
